Weather & House Price Prediction Project

Overview

This is a simple Python-based project that combines:

- Real-time weather data using an API
- Basic house price estimation logic

The goal of this project is to demonstrate API integration and basic problem-solving using Python.

---

Features

- Get live weather details (temperature, humidity, condition)
- Estimate house price based on area and number of bedrooms

---

Technologies Used

- Python
- Requests library
- OpenWeather API

---

How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the program:

python main.py

---

Example

Weather:

Enter city: Delhi
Output: Delhi → Temp: 32°C, Humidity: 60%, Condition: clear sky

House Price:

Enter area: 1000
Enter bedrooms: 2
Output: Estimated House Price: ₹4000000
